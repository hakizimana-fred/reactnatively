# User Profile

**Role:** Lead developer building a production-grade React Native design system library.

**Expertise:** Experienced — comfortable with monorepo tooling (Turborepo, pnpm), TypeScript generics, React Native architecture, design systems.

**Style:** Demands production-ready implementations. Explicitly said "Do NOT stay at theory level. Start implementing." — no half-measures, no scaffolding, no TODO stubs.

**Expectations:**
- World-class TypeScript DX (every prop typed, no `any` in library code)
- Glass-first architecture: every component should optionally render glass backing
- Animation-first: press, entrance, gesture animations are non-negotiable
- Respects established patterns: defineVariants for O(1) lookups, compound components, pub-sub for imperative APIs
